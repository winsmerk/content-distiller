# 单视频蒸馏任务流程

> 适用场景：输入一条小红书视频链接，抓取该视频公开数据、评论与口播，生成单视频深度蒸馏报告和创作指南 Skill。

## 任务目标

给定一条小红书视频链接，完成：

1. 解析短链，获取小红书 note_id。
2. 通过 TikHub API 拉取视频详情、互动数据和评论。
3. 可选开启 Whisper，提取视频口播逐字稿。
4. 生成最终产物：
   - `output/{视频名称}_视频蒸馏报告.html`
   - `output/{视频名称}_视频创作指南.skill/SKILL.md`
   - `output/_过程文件/原始素材/{视频名称}_视频AI蒸馏任务.md`

## 前置准备

进入项目目录并激活虚拟环境：

```bash
cd /Users/lww/Desktop/project/deepdata/blogger-distiller
source .venv/bin/activate
```

建议 Token 使用配置文件或环境变量，不要写入仓库文件：

```bash
export TIKHUB_API_TOKEN="你的TikHubToken"
```

## 一键执行

```bash
python run_video.py "小红书视频链接或整段分享文案"
```

示例：

```bash
python run_video.py "Codex➕Obsidian•10min搭爆款🔥工作流教程🙌 http://xhslink.com/o/8jmW9E4YOAx" --name "Codex_Obsidian_工作流"
```

默认会开启口播转写。如需跳过：

```bash
python run_video.py "http://xhslink.com/o/8jmW9E4YOAx" --no-transcript
```

如果短链无法解析，可手动指定 note_id：

```bash
python run_video.py "原始链接" --note-id "小红书note_id"
```

## 分步执行

### Step 1：采集单视频

```bash
python scripts/crawl_xhs_video.py "http://xhslink.com/o/8jmW9E4YOAx" --name "Codex_Obsidian_工作流"
```

生成：

```text
data/Codex_Obsidian_工作流_video_details.json
data/Codex_Obsidian_工作流_video_meta.json
```

### Step 2：生成最终报告和 Skill

```bash
python scripts/generate_video_final.py data/Codex_Obsidian_工作流_video_details.json --name "Codex_Obsidian_工作流"
```

生成：

```text
output/Codex_Obsidian_工作流_视频蒸馏报告.html
output/Codex_Obsidian_工作流_视频创作指南.skill/SKILL.md
output/_过程文件/原始素材/Codex_Obsidian_工作流_视频AI蒸馏任务.md
```

## 使用方式

HTML 报告用于人工阅读：

```bash
open output/Codex_Obsidian_工作流_视频蒸馏报告.html
```

Skill 用于创作同类视频内容。在 Codex 中可以这样说：

```text
请加载 output/Codex_Obsidian_工作流_视频创作指南.skill/SKILL.md，
帮我写一条「AI 工具工作流教程」的小红书视频脚本。
```

## 注意事项

- 单视频分析只代表这条视频本身，不能推断博主长期账号策略。
- 口播转写依赖 Whisper 和 ffmpeg；若不可用，仍会基于标题、正文、互动和评论生成报告。
- 小红书短链如解析失败，可以复制 App 内原始链接或手动提供 note_id。
- 评论者身份会在写入本地 JSON 前脱敏为“读者N / 作者”。
